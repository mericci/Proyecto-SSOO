La estructura del proyecto contiene una carptea con todos los modulos exigidos por el proyecto, esta carpeta se llama **crfs** y contiene a **main.c**, **cr_API.c** y **cr_API.h**. Por otro lado se tiene la carpeta **util** que posee las funciones que son utiles y que complementan la programación, en esta carpeta se encunetran los archivos **util.c** y **util.h**.


La documentación del proyecto se encuentra en el archivo DocumentacionProyectoSSOO.pdf.
Para el levantamineto de errores, se les asigno un nombre a cada uno según criterio del grupo.
